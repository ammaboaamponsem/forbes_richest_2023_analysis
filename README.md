# Forbes 2023 Wealth Distribution Analysis

## 📊 Overview
An analytical breakdown of the 2023 Forbes list to understand global wealth distribution, industry-specific trends, and economic concentration.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Seaborn

## 📈 Key Findings
- Analyzed wealth concentration across industries and geographies.
- Visualized top wealth generators in the global market.
